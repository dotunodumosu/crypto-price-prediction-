# Cryptocurrency Price Prediction using Machine Learning

## 📌 Project Overview
This project focuses on predicting Bitcoin prices using machine learning models. The goal is to analyze historical data and build models that can forecast future prices.

## 📊 Dataset
The dataset was collected using Yahoo Finance (BTC-USD historical data).

## ⚙️ Methods Used
- Data Cleaning
- Feature Engineering (Moving Averages, Returns)
- Machine Learning Models:
  - Linear Regression
  - Random Forest

## 📈 Results
The models were evaluated using:
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)

Random Forest performed better in predicting Bitcoin prices compared to Linear Regression.

## 📉 Visualization
The project includes graphs comparing actual vs predicted prices.
